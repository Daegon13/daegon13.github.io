resumen: "es una web corporativa desarrollada para una empresa de transporte y logística que ofrece servicios de envíos nacionales e internacionales, gestión de flotas y coordinación logística personalizada.  
El objetivo del proyecto fue crear una interfaz moderna, funcional y adaptable a distintos dispositivos, enfocada en transmitir **seriedad, confianza y eficiencia**."
El sitio incluye:
- Página principal con presentación de servicios, misión y valores.  
- Sección de contacto conectada a **Firebase** para registro automático de consultas.  
- Diseño **responsive** optimizado para clientes móviles.  
- Integración de panel administrativo para la gestión interna de servicios.


El desarrollo priorizó la **experiencia del usuario**, el **rendimiento** y la **escalabilidad**, asegurando una estructura limpia, modular y lista para futuras ampliaciones.
